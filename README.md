

# Some-Quotes
```
An API to fetch quotes by sending a GET request.
```

## API Reference

#### Get all items

```http
  GET /quote
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/SyntaxErrorSolos/some-quotes
```

Go to the project directory

```bash
  cd some-quotes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node .
```
